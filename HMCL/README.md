## 教程简介

你将学会怎么使用HMCL启动器游玩Minecraft并且和你的好朋友一起联机

1. Java下载，安装，设置环境变量
2. HMCL启动器下载，设置启动器，下载Minecraft
3. LittleSkin注册，设置，皮肤，披风
4. HMCL启动器自带的联机功能
5. 内外穿透联机
